# Informazioni della cartella BG	                                           #
################################################################################

Le immagini presenti in questa cartella saranno quelle che scorrono come sfondo di ogni pagina

Background: il nome formato da bg_<progressivo int(2)>.jpg
e.g.
	bg_01.jpg
Tali immagini devono avere le dimensioni di (1920px,1200px).

ATTENZIONE: Le foto devono essere salvate in formato JPEG (.jpg), prestare attenzione alla codifica.
			Non cancellare ne modificare l'immagine "Home-BG.jpg".
