# Informazioni della cartella Concorsi	                                       #
################################################################################

Le immagini presenti in questa cartella saranno quelle presenti nella sessione concorsi

Vi sono tre tipi di immagini (obbligatorie):

1) Menu: il nome formato da concorso_<AAAA>_<nome>_menu.jpg, tali foto andranno a collocarsi nel menu concorsi di ogni pagina
e.g.
	concorso_2013_esempio_menu.jpg
Tali immagini devono avere le dimensioni di (370px,200px).

2) Locandina: il nome formato da concorso_<AAAA>_<nome>_menu_locandina.jpg, tali foto andranno a collocarsi nella locandina dei concorsi
e.g.
	concorso_2013_esempio_menu_locandina.jpg
Tali immagini devono avere le dimensioni di massimo (1450px,2050px).

3) Principale: il nome formato da concorso_<AAAA>_<nome>_menu.jpg, tali foto saranno quelle principali del concorso
e.g.
	concorso_2013_esempio_menu.jpg
Tali immagini devono avere le dimensioni di (868px,410px).

ATTENZIONE: Le foto devono essere salvate in formato JPEG (.jpg), prestare attenzione alla codifica.
