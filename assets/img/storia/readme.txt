# Informazioni della cartella Storia  	                                       #
################################################################################

Le immagini presenti in questa cartella saranno quelle linkate e caricate
nella sessione storia.

Vi sono due tipi di immagini (obbligatorie):

1) Menu: il nome formato da storia_<AAAA>_menu.jpg, tali foto andranno a collocarsi nel menu storia a comparsa (larga)
e.g.
	storia_2014_menu.jpg
Tali immagini devono avere le dimensioni di (868px,410px).

2) Storia: il nome formato da storia_<AAAA>.jpg, tali foto andranno a collocarsi nel menu storia a comparsa (piccola) e nella pagina storia
e.g.
	storia_2014.jpg
Tali immagini devono avere le dimensioni di (156px,156px).

ATTENZIONE: Le foto devono essere salvate in formato JPEG (.jpg), prestare attenzione alla codifica.
